解决办法

根据这篇解答，github - git pull encounters kex_exchange_identification: Connection closed by remote host - Stack Overflow，我在~/.ssh/config后面加上了

1
2
3

Host github.com
 Hostname ssh.github.com
 Port 443

并将Clash调整为TUN Mode，才可以成功。（System Proxy还是不行）
